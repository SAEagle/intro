Unreal Arena 3 - nostalgia project

The idea behind the project is to move Quake 3 Arena to UE 5. I've always wondered how it would be to play Quake with UE. In future planning add netcode to support local multiplayer. Here is the link to the repo where you can check the latest updates: https://github.com/SAEagle/unrealarena3

Challenges:

  • C++ core of the project with minor decorators in the blueprints
  • Q3A assets migration to Unreal 5
  • Moving all vfx to Niagara System

ASH

This project was built for the 3-4 months in my spare time. The whole logic, animation states, level design, UI, and materials were done in blueprints. Unique stances combat mechanic with 3 stances for defense and attack. The initial project was used for fundraising and support later development. The project: Demon Skin. It was an interesting period of researchinh the 3d sidescrollers themes with interesting combat mechanic.

Challenges:

  • Root Animations implementation and matching it with states
  • Whole inventory system and the UI crafted in the Blueprints

Hovecraft Shooter

Arcade shooter with strategy elements. Inspired by HIGHFLEET.

At the beginning of the 20th century, ether was discovered, which replaced internal combustion engines. People have mastered gravity and after the world war, the world is in decline. You, as a representative of one of the governors of the current region, will have to restore order in the wastelands...

The goal was to recreate a HIGHFLEET on the hovercrafts.

Challenges:

  • Physics calculations for hovercrafts
  • Trade system with trading UI
  • Open world map with different map events

Realtime RPG Combat

The purpose of this project was to recreate a feeling of the realtime decision making in a grid based combats. The 3 levels with different enemy types can show the basics of the mechanic.

The goal was to create realtime combat on the grid based map.

Challenges:

  • Realtime combat logic based on the timers
  • Grids and the AI logic

Space Exploration

Classic JRPG-style combat. Explore the solar system map on your own ship. Onboard the enemy ships and conquer them. Open real-time objects on the map with indoor Turn Based combat.

The goal was to recreate JRPG style combat.

Challenges:

  • Turn Based combat mechanics in blueprints
  • Simple scalable grid for the levels
